# API Mistral for OCR

## Presentation 

Fast OCR test pipeline with Mistral API OCR.

## Step by step :

1. Put you're image on the folder, be sure the image_path variable have the same name
2. Generate an API Key on https://admin.mistral.ai/organization/api-keys
3. create a new environnement 
4. add the API key to you're environnement variable with the name MISTRAL_API_KEY with `export MISTRAL_API_KEY="YOUR KEY"`
5. execute `pip install mistralai` in you're terminal
6. execute `python run API.py`
